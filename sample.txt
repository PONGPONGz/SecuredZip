This is a sample text file to demonstrate the Secure Zip application.

The application will compress this text using the zlib algorithm and then encrypt it
using AES-256 encryption in GCM mode. This provides both strong compression and
military-grade encryption.

Features of the Secure Zip application:
1. Text compression reduces file size
2. AES-256 encryption provides strong security
3. Password-based encryption ensures only authorized users can access the content
4. Both command-line and graphical interfaces are available
